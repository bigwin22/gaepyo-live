# gaepyo-live

Korea election live results hub. 2026년 6월 3일 제9회 전국동시지방선거 개표 상황을 빠르게 확인하기 위한 GitHub Pages 정적 사이트입니다.

## 목표

- 중앙선거관리위원회 선거통계시스템의 공개 개표 표를 기준으로 지역별 개표율과 후보 경합도를 보여줍니다.
- 선관위 공식 개표 페이지, 개표소 CCTV, 주요 방송사 선거 라이브를 한 화면에서 바로 열 수 있게 합니다.
- 서버 없이 GitHub Pages에서 동작하도록 HTML, CSS, JavaScript만 사용합니다.

## 개발 방식

변경 이력이 잘 남도록 기능 단위로 커밋합니다.

1. 정적 사이트 초기 구조
2. 선관위 개표 데이터 호출/파싱
3. 지역 검색 및 후보 경합 UI
4. 방송/공식 링크 정리
5. GitHub Pages 배포 설정
